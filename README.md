# CInbora Entrar no CIn - CIn Open Day 50 anos

## Descrição

CInbora é um projeto dos alunos Fábio Filho, Lucas Azevedo e Sara Araujo, do curso de Sistemas de Informação do Centro de Informática da Universidade Federal. CInbora foi um projeto voltado aos estudantes do ensino médio que estão na fase de realizar vestibular e fornece uma visão sobre o Centro de Informática da UFPE.
